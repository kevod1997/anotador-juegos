# Reglamento de Juegos de Dados

Este documento contiene las reglas oficiales para **La Generala** y **El 10.000**, transcritas de las instrucciones proporcionadas.

---

## 1. La Generala

**Edad:** A partir de 5 años  
**Jugadores:** 2 o más  
**Componentes:** 5 dados, un cubilete, block de hojas para anotar.

### Objetivo
Obtener la mayor puntuación posible al sumar los números y los juegos mayores al finalizar las rondas.

### Preparación
Para decidir quién comienza, cada jugador arroja un dado. El que obtenga el mayor puntaje será el primero.

### Cómo Jugar
* La partida se desarrolla en **11 rondas**.
* Cada jugador tiene un turno por ronda.
* En cada turno, el jugador tiene **3 tiros**.
* Luego de cada tirada, el jugador puede decidir qué dados desea conservar y volver a tirar el resto para buscar una combinación.
* Al finalizar sus tiros (o antes si está satisfecho), debe elegir una categoría en la que anotar los puntos obtenidos.
* Si no logra ninguna combinación válida para una categoría vacía, deberá tachar una (anotar 0 puntos).

### Sistema de Puntuación

#### A. Por Número (Categorías 1 al 6)
Se calcula multiplicando la cantidad de dados que muestran ese número por el propio número.
* *Ejemplo:* Tres dados con el número 6 = $3 \times 6 = 18$ puntos.

#### B. Juegos Mayores
Estas combinaciones otorgan puntajes fijos.

| Combinación | Descripción | Puntos |
| :--- | :--- | :--- |
| **Escalera** | 1-2-3-4-5 ó 2-3-4-5-6 | **20** |
| **Full** | 3 dados de un número y 2 de otro | **30** |
| **Póker** | 4 dados iguales | **40** |
| **Generala** | 5 dados iguales | **50** |
| **Doble Generala** | Igual a la anterior (es optativa) | **100** (aprox/varía) |

### Reglas Especiales

1.  **Juego Servido:** Se produce cuando, al arrojar los dados una sola vez (en el primer tiro del turno), un jugador obtiene un juego mayor.
    * **Premio:** Se agregan **5 puntos extra** al valor de la jugada (excepto en la Generala).
2.  **¡Importante! (Generala Servida):** Si un jugador obtiene Generala Servida (5 dados iguales en el primer tiro), **gana automáticamente la partida**.

### Final de la Partida
La partida termina cuando un jugador obtiene una Generala Servida o cuando todos los jugadores han completado las 11 rondas (llenando todas las casillas). En este último caso, se suman los puntajes y el jugador con más puntos gana.

## 2. El 10.000

**Jugadores:** Mínimo 2.  
**Elementos:** 5 dados.

### Objetivo
Ser el primer jugador en alcanzar un total exacto o superior a **10.000 puntos**.

### Cómo Jugar
Por turnos, cada jugador tira los 5 dados. Para sumar puntos se necesita sacar al menos:
* Un 1.
* Un 5.
* Tres dados iguales.
* Una escalera.

**Mecánica de "Plantarse" o "Seguir":**
1.  Si el jugador obtiene dados que suman puntos, puede apartarlos y sumar ese puntaje parcial.
2.  Luego decide:
    * **Plantarse:** Anota los puntos acumulados en ese turno y pasa los dados al siguiente jugador.
    * **Seguir tirando:** Tira los dados restantes para mejorar su puntuación.
3.  **Riesgo:** Si elige continuar y en el nuevo tiro **no logra ninguna combinación válida**, pierde **todos** los puntos acumulados en ese turno y le toca al siguiente jugador.
4.  **Tirar los 5 dados de nuevo:** Si un jugador lanza una combinación que utiliza los 5 dados (y suma puntaje con todos), puede seguir tirando con los 5 dados nuevamente, acumulando más puntos en el mismo turno.

### Tabla de Puntos

| Combinación | Valor |
| :--- | :--- |
| **Cada 1** | 100 puntos |
| **Cada 5** | 50 puntos |
| **Tres dados iguales** | Valor del dado x 100 (Ej: tres 4 = 400 pts) |
| **Tres dados de 1** | 1.000 puntos |
| **Escalera** (1-2-3-4-5 o 2-3-4-5-6) | 500 puntos |
| **Cinco dados iguales** | **10.000 puntos (Gana la partida)** |

### Ganador
El primer jugador en alcanzar un total de 10.000 puntos gana la partida inmediatamente.
